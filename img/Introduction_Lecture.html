motivation

things to learn

    • obtain data

    • do data-project management

    • use a (simple) neural network

Rules of the game

to learn python we use datacamp and lectures:

  • you need to finish the datacamp courses in time
  • otherwise you will be lost during the lectures
  • see the Lecture schedule for the deadlines
  • prepare the lectures by going through the notebook beforehand
  • participate actively during the lectures
  • you do not learn datascience by passively typing in answers!
  • for the second part of the course: there are both datacamp courses and a (free online) textbook

to get a grade for the python track:

  • finish Assignment 1 before the deadline
  • Final Assignment
  • check the deadlines for the final assignment
